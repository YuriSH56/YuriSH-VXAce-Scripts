# YuriSH's RPGMaker VXAce Scripts
This repository holds all of my LISA scripts.\
"Scripts" folder includes source code for scripts.\
"Assets" folder includes assets required for some of the scripts.\
"Fixes" subfolder in "Scripts" holds scriptlets that fix some of the issues i've noticed in other scripts not made by me.

Feel free to open an issue if you find any errors in the scripts or if script don't work at all.
